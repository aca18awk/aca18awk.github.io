Taxi Website

Date October 2018 – May 2019
Description This was the biggest University project I've worked on. I collaborated in a team of 7 to design and develop fully functioning software product for Taxi Company called 'Tweet a Ride'. It was designed to help them manage the customers, cars and all the tweets where users ordered a ride.
Results First Class Mark
How it works On main webpage everyone could see the offer of Taxi Company (called 'Tweet a Ride'), create an account and log in.
Logged Customers could see all the avaiable promotions, history of rides and their account details.
Logged Admins could see database of Customers, Cars (available and currewntly taken) and other Admins and manipuate this data. They could also add new avaiable promotions for individual and all users. Admin could also manage all the tweets which mentioned the Tweet a Ride company on twitter to order a ride.
System automatically got all necesarry data and asked if anything was missing. System also automatically added promotions to the user's accounts.

I wrote most of Front-end code with CSS, HTML and JavaScript.
I also worked on Back-End using Ruby and I was responsible for Databases (SQLite).
We made sure that all the data is secured properly by using hashing algorithms.
Having developed the system, we spent a lot of time testing it (both manually and using testing tools like Cucumber).
