Bombe Symulator

Date December 2019
Description As final assignment of Functional Programming, we had to write a program which simulated Bombe, machine which broke Enigma.
We were asked to followed the specified design and we assumed (for simplicity) that the choice of rotors and reflector in Bombe is always the same.

As an extention we could experiment with simulation. I decided to to create the algorithm which would take approximately the same amount of time to run as the initial Bombe algorithm but it would produce correct results much often.
Results First Class Mark (100%).

Those are a few examples which compare runtime and corectness Bombe we were asked to code and Extended Bombe I made.
