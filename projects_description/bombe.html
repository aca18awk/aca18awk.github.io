Bombe Simulator

Date December 2019
Description As a final assignment of Functional Programming, we had to write a program which simulated a Bombe - machine which broke Enigma.
We were asked to follow the specified design and we assumed (for simplicity) that the choice of rotors and reflector in Bombe is always the same.

As an extension we could experiment with simulation. I decided to create the algorithm which would take approximately the same amount of time to run as the initial Bombe algorithm but it would produce correct results much often.
Results First Class Mark (100%).
