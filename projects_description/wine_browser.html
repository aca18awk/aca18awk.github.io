Portugese Wine Browser

Date April - May 2019
Description This is project I had to make as final Java Programming Assignment.
The basics of the system were written by my lecturer.
I built an Interactive GUI browser that allows wine producers explore the quality of their wine (2 types), based on the wine samples previously collected by them (6499).
I made use of OO programming, the Java Collections Framework, Java Swing and event handling for the GUI-building.
Results First Class Mark.
